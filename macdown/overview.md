# MacDownToo — Overview

A modern, lightweight Markdown editor for macOS. Private fork/continuation of MacDown.

## Repo

`dglcinc/MacDownToo` (private) — [github.com/dglcinc/MacDownToo](https://github.com/dglcinc/MacDownToo)

Local clone: `~/github/MacDownToo`

## What it is

Native macOS (Objective-C/Cocoa) markdown editor with live split-pane preview.
Continuation of the MacDown lineage (Mou → MacDown → MacDown 3000 → MacDownToo).
Free, MIT licensed, universal binary (Apple Silicon + Intel).

## Key features

- Live preview via Hoedown markdown renderer
- Prism syntax highlighting in code blocks
- GFM: tables, strikethrough, task lists, autolinks
- Math notation (TeX: `$$...$$`, `\[...\]`)
- Jekyll front-matter support
- Export to HTML or PDF
- Editor and preview themes from Mou heritage
